Example

Basic usage

A very simple example.

<Button token="123" countryCode="AU" />

Logo

Display text logo, image logo or both.

<Button token="123" countryCode="AU">
  Buy now with
</Button>
<label> </label>
<Button token="123" countryCode="AU" logo="image">
  Buy now with
</Button>
<label> </label>
<Button token="123" countryCode="AU" logo="text">
  Buy now with
</Button>

Theme

Different button styles. white-black is the default style.

<Button token="123" countryCode="AU" theme="white-black">
  Shop now with
</Button>
<label> </label>
<Button token="123" countryCode="AU" theme="mint-black">
  Shop now with
</Button>
<label> </label>
<Button token="123" countryCode="AU" theme="black-mint">
  Shop now with
</Button>

Clearpay

When the countryCode='GB', the logo will be Clearpay.

<Button token="123" countryCode="GB">
  Shop with
</Button>
