# Usage

## Examples

### Basic usage

A very simple example.

<Playground>
	<Button token="123" countryCode="AU" />
</Playground>

### Logo

Display text logo, image logo or both.

<Playground>
	<Button token="123" countryCode="AU">Buy now with</Button><label> </label>
	<Button token="123" countryCode="AU" logo="image">Buy now with</Button><label> </label>
	<Button token="123" countryCode="AU" logo="text">Buy now with</Button>
</Playground>

### Theme

Different button styles. `white-black` is the default style.

<Playground>
	<Button token="123" countryCode="AU" theme="white-black">Shop now with</Button><label> </label>
	<Button token="123" countryCode="AU" theme="mint-black">Shop now with</Button><label> </label>
	<Button token="123" countryCode="AU" theme="black-mint">Shop now with</Button>
</Playground>

### Clearpay

When the `countryCode='GB'`, the logo will be Clearpay.

<Playground>
	<Button token="123" countryCode="GB">Shop with</Button>
</Playground>


## API

| Property    | Description                                                 | Type                                | Default |
| ----------- | ----------------------------------------------------------- | ----------------------------------- | ------- |
| token       | (required) checkout token to be used to complete payment.   | string                              | -       |
| countryCode | (required) country code.                                    | string                              | -       |
| behavior    | method of launching the checkout page.                      | enum: `popup` &#124; `redirect`     | `popup` |
| onComplete  | callback when transaction is complete. **popup mode only**. | (event: any) => void                | -       |
| logo        | determine how to display the logo.                          | enum: `image` &#124; `text` &#124; `all`  | `all`  |
| theme       | color of button's background and label.                     | enum: `white-black` &#124; `black-mint` &#124; `mint-black` | `white-black` |
| className   | classname of button wrapper.                                | string                              | -       |
| style       | extra styles.                                               | React.CSSProperties                 | -       |
